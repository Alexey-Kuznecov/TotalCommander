NL_Info, Версия 1.20
  Информационный плагин для программы Total Commander, позволяющий отображать информацию
  о жёстких и символических ссылках NTFS.

Автор плагина: Власов Константин, 2011 г.
Домашняя страница: http://flint-inc.ru/
E-mail:            support@flint-inc.ru

Распространяется по лицензии BSD 2-Clause.

При написании плагина использовался исходный код программы Junction.exe.
Автор программы - Mark Russinovich (http://www.sysinternals.com/).


Описание
--------

Плагин содержит следующие поля данных:
1. Число жёстких ссылок [Number of Hard Links]
   Счётчик жёстких ссылок данной файловой записи. Для обычных файлов (а также для файлов
   на разделах не-NTFS) равен 1, т.к. сам файл - это тоже жёсткая ссылка.
   Это поле доступно только для файлов.
2. Тип символической ссылки [Reparse Point Type]
   Символические ссылки бывают разные. Вот полный список определяющихся типов:
     Junction (обычная ссылка; в Vista/7 называется точкой соединения)
     Mount Point (точка монтирования раздела)
     Symbolic Link (новый тип символических ссылок, появившихся в Windows Vista)
     Hierarchical Storage Management Reparse
     Native Structured Storage Point
     Native Structured Storage Recovery Point
     Single Instance Store Point
     Distributed File System Point
     Unknown Microsoft Reparse Point
     Third-Party Reparse Point
3. Тип символической ссылки (расш) [Reparse Point Type (Ext)]
   Это поле выводит то же самое, что и "Тип символической ссылки", но дополнительно для
   простых каталогов выдаётся значение "Folder", а для файлов - "File". Это может быть
   полезно, например, для поиска или выделения только каталогов, но не символических
   ссылок.
4. Целевой объект [Reparse Point Target]
   Объект, на который указывает символическая ссылка. Для точек монтирования разделов
   плагин автоматически преобразует идентификатор тома, хранящийся в файловой записи, в
   обычное буквенное обозначение раздела (разумеется, если этому разделу назначена буква
   диска).
5. Относительная ссылка [Relative]
   Флаг-индикатор, указывающий, является ли ссылка абсолютной или относительной. Применимо
   только к ссылкам типа Symbolic Link.
6. Корректная ссылка [Valid Target]
   Выдаёт значение "Да" или "Нет" в зависимости от того, доступен ли целевой объект
   данной символической ссылки.


Системные требования
--------------------
32-битная версия:
	Windows 2000 и выше
	Total Commander 7.50–8.0 x32

64-битная версия:
	Процессор с поддержкой 64-битных команд (Intel Pentium 4, AMD Athlon 64 и выше)
	Windows XP x64 и выше
	Total Commander 8.0 x64
