PhotoViewer 0.9.4.6 Beta
Plugin for TC for view images.
This is 64-bit replacement basic functions of SGViewer plugin (http://totalcmd.net/plugring/sgviewer.html).
Supported formats: BMP, JPEG, GIF, TIFF, PNG, PSD, JPEG2000, EMF, WMF, DICOM, Camera RAW. Animation not supported.

Some functions in beta version for test only and may absent in next releases.
