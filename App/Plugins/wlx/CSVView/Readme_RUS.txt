CSVView 0.9.3.2 Beta

Плагин для просмотра файлов CSV.
Это новая независимая версия CSV Viewer (http://wincmd.ru/plugring/wlx_csv.html).


*** Установка ***

При попытке открыть архив с плагином TC выдаст сообщение об установке плагина.


*** Условия распространения ***

  Это программное обеспечение (ПО) распространяется по принципу "КАК ЕСТЬ" ("AS IS") и является бесплатным для коммерческого и некоммерческого использования. Автор не дает никаких гарантий по работе этого ПО и не несет никакой ответственности при его использовании.


*** История изменений ***

Версия 0.9 Beta
 * публичная версия.

Версия 0.9.1 Beta
 + поиск;
 * по умолчанию непечатаемые символы игнорируются (IgnoreSpecialCharacters и ConvertTabSpecialCharacter в ini);
 * различные исправления.

Версия 0.9.2 Beta
 + показ номеров строк (ShowRowNumber в ini, по умолчанию 1);
 * из-за возможных проблем обновление при парсинге отключено (LiveUpdate в ini, по умолчанию 0);
 * различные исправления.

Версия 0.9.3 Beta
 + сортировка с учётом цифровых значений (NumericSort в ini, по умолчанию 0);
 * различные исправления.

Версия 0.9.3.1 Beta
 + настройки шрифта (FontName и FontSize в ini).

Версия 0.9.3.2 Beta
 * исправлена ошибка при появлении символа с кодом 0 (null).
