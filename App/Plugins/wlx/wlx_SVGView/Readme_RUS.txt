SVGView 1.2.7
Плагин для TC для просмотра файлов SVG.

Плагин может использовать RSVG (rsvg-convert.exe) или ImageMagic для конвертирования файлов SVG в растр.

Распакованная версия RSVG (с сайта http://opensourcepack.blogspot.ru/2012/06/rsvg-convert-svg-image-conversion-tool.html) используется по умолчанию и включена в архив с плагином.
Для загрузки SVG с юникодными именами плагин копирует такие файлы во временную папку и открывает их.
Это поведение можно отключить (такие файлы открываться не будут):
[Setup]
UseRSVGHelper=0

Для использования конвертера ImageMagick нужно скачать его с сайта
http://www.imagemagick.org/script/binary-releases.php#windows
Файлы нужно поместить в подпапки в каталоге плагина:
ImageMagick_x32\
для 32-битной версии ImageMagick, и
ImageMagick_x64\
для 64-битной версии ImageMagick и отключить использование RSVG:
[Setup]
UseRSVG=0

История изменений

Версия 1.2.1:
 * публичная версия.

Версия 1.2.2:
 * новая версия rsvg-convert.exe;
 + OpenWidth= в секции [Setup] INI-файла: устанавливает ширину для загружаемого изображения (10-10000, 0 по умолчанию);
 + OpenHeight= в секции [Setup] INI-файла: устанавливает высоту для загружаемого изображения (10-10000, 0 по умолчанию).

Версия 1.2.3:
 * исправлен баг с изображениями в режиме эскизов TC.

Версия 1.2.4:
 * для юникодных имен сначала делается попытка получить короткое имя файла;
 * исправлено возможное падение при размере файла 0 байт.

Версия 1.2.5:
 + файлы с расширением svgz могут быть архивами zip и содержать файл svg с таким же именем;
 + RSVGPath= в секции [Setup] INI-файла: устанавливает полный путь к конвертеру rsvg.

Версия 1.2.7:
 * исправление для попытки получить короткое имя файла (для юникодных и очень длинных имен);
 + пункт меню "Reload Image" для перезагрузки изображения;
 * незначительные внутренние изменения.
