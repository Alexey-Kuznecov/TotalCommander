Установка:

  1. Во первых, скачать архив и распаковать.
  2. Содержимое архива распакуте в папку с модами The Witcher 3\Mods (если она не существует, то просто создайте ее).
  3. Запустить игру.
Конфликты:

    Если вы используете много модов, возможны конфликты. Ниже представлены 3 способа решения данной проблемы:

  1. Пройти по пути и удалить там файлы btHack.ws и btTaskMaintainSpeed.ws:

    The Witcher 3 Wild Hunt\content\content0\scripts\game\behavior_tree\tasks \hacks

  2. Установить Witcher 3 Mod Scripting Fix: ссылка. Просто поместить папку mod0000_0TelemetryKeyword в папку Mods.
  3. Если они возникают, то можно попробовать использовать Script Merger — инструмент для слияния скриптов, он может помочь решить возникшую проблему.

Удалить:

  • Удалите распакованную папку из директории The_Witcher_3_Wild_Hunt/Mods/.
