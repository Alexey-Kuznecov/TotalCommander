Встроенные команды с поддержкой параметров

Просмотр

  

cm_List имя_файла

Внутренний просмотр указанного файла (при любых настройках просмотра). До версии 9.0 это была единственная из встроенных команд TC, способная работать с параметрами. Формат записи команды в этом случае — только строковый

Задержка исполнения

  

cm_wait время_ожидания

Ожидать в течение NN миллисекунд. В качестве параметра может быть указан также символ ? для открытия диалога с запросом значения

Выделение

  

cm_Select число_файлов/каталогов

Параметр указывает число объектов, начиная с текущего, которые будут выделены; курсор переместится на следующий объект после этой выборки. С параметром ? или ?число здесь и далее выполнению команды предшествует вывод диалога с запросом значения

cm_UnSelect число_файлов/каталогов

Параметр указывает число объектов, с которых будет снято выделение; курсор переместится на следующий объект после этой выборки

cm_Reverse число_файлов/каталогов

Параметр указывает число объектов, у которых будет инвертировано выделение; курсор переместится на следующий объект после этой выборки

Навигация

  

cm_GoToFirstEntry номер_строки

Числовой параметр, прибавленный к 1, указывает, на строку какого по порядку объекта (файла/каталога) будет установлен курсор

cm_GoToFirstFile номер_строки

Числовой параметр, прибавленный к 1, указывает, на строку какого по порядку файла (не каталога!) будет установлен курсор

cm_SrcOpenDrives имя_диска

Открытие диска в активной панели. Параметр — буква диска (A-Z), а также / (прямой слэш) для корня диска, . (точка) для родительского каталога и [ (открывающая квадратная скобка) для панели сетевого окружения). Для скриптов используйте код символа, например, A=65

cm_LeftOpenDrives имя_диска

Открытие диска в левой панели. Параметр — буква диска (A-Z), а также / (прямой слэш) для корня диска, . (точка) для родительского каталога и [ (открывающая квадратная скобка) для панели сетевого окружения). Для скриптов используйте код символа, например, A=65

cm_RightOpenDrives имя_диска

Открытие диска в правой панели. Параметр — буква диска (A-Z), а также / (прямой слэш) для корня диска, . (точка) для родительского каталога и [ (открывающая квадратная скобка) для панели сетевого окружения). Для скриптов используйте код символа, например, A=65

cm_SearchForInCurdir имя_диска

Поиск файлов на указанном диске. Параметр — буква диска (A-Z). Для скриптов используйте код символа, например, A=65

Файловые панели

  

cm_VerticalPanels число

Расположить панели одну над другой. Параметр — 0 (как без параметра, с переключением режимов), 1/2 (только включение/выключение режима)

cm_100percent число

Установить размеры панелей. Параметр — положительное число: ширина активной панели, отрицательное число: ширина левой панели. Пример: cm_100percent 70

Режимы представления

  

cm_ScrThumbs число

Просмотр эскизов в активной панели. Параметр — 0 (как без параметра, с переключением режимов), 1/2 (только включение/выключение эскизов)

cm_LeftThumbs число

Просмотр эскизов в левой панели. Параметр — 0 (как без параметра, с переключением режимов), 1/2 (только включение/выключение эскизов)

cm_RightThumbs число

Просмотр эскизов в правой панели. Параметр — 0 (как без параметра, с переключением режимов), 1/2 (только включение/выключение эскизов)

cm_LeftSwitchToThisCustomView номер_набора

Числовой параметр задаёт номер набора колонок, открываемого в левой панели

cm_RightSwitchToThisCustomView номер_набора

Числовой параметр задаёт номер набора колонок, открываемого в правой панели

Сортировка

  

cm_SrcByName число

Сортировка по имени в активной панели. Параметр — 0 (как без параметра, с переключением направления), 1/2 (только прямая/обратная).Также для этой и иных подобных команд есть параметр 4 (сортировка по дополнительной колонке), суммирующийся с основным параметром

cm_LeftByName число

Сортировка по имени в левой панели. Параметр — 0 (как без параметра, с переключением направления), 1/2 (только прямая/обратная)

cm_RightByName число

cm_SrcByExt число

cm_LeftByExt число

Сортировка по типу в левой панели. Параметр — 0 (как без параметра, с переключением направления), 1/2 (только прямая/обратная)

Сортировка по типу в правой панели. Параметр — 0 (как без параметра, с переключением направления), 1/2 (только прямая/обратная)

cm_SrcBySize число

Сортировка по размеру в активной панели. Параметр — 0 (как без параметра, с переключением направления), 1/2 (только прямая/обратная)

Сортировка по размеру в левой панели. Параметр — 0 (как без параметра, с переключением направления), 1/2 (только прямая/обратная)

cm_SrcByDateTime число

Сортировка по дате/времени в активной панели. Параметр — 0 (как без параметра, с переключением направления), 1/2 (только прямая/обратная)

Сортировка по дате/времени в левой панели. Параметр — 0 (как без параметра, с переключением направления), 1/2 (только прямая/обратная)

cm_RightByDateTime число

Сортировка по дате/времени в правой панели. Параметр — 0 (как без параметра, с переключением направления), 1/2 (только прямая/обратная)

cm_SrcNegOrder число

Сортировка в обратном порядке в активной панели. Параметр — 0 (как без параметра, с переключением направления), 1/2 (только вкл/выкл)

cm_LeftNegOrder число

Сортировка в обратном порядке в левой панели. Параметр — 0 (как без параметра, с переключением направления), 1/2 (только вкл/выкл)

cm_RightNegOrder число

Сортировка в обратном порядке в правой панели. Параметр — 0 (как без параметра, с переключением направления), 1/2 (только вкл/выкл)

Стили оформления

  

cm_SrcViewModeList число/название

Переключение в активной панели на указанный стиль оформления. Параметр — порядковый номер стиля в списке (положительное число, работает везде) или его пользовательское название (не работает в командной строке)

cm_LeftViewModeList число/название

Переключение в левой панели на указанный стиль оформления. Параметр — порядковый номер стиля в списке (положительное число, работает везде) или его пользовательское название (не работает в командной строке)

cm_RightViewModeList число/название

Переключение в правой панели на указанный стиль оформления. Параметр — порядковый номер стиля в списке (положительное число, работает везде) или его пользовательское название (не работает в командной строке)
